Featured Projects#

bunshin-go#

Status: Active Development · Stack: Go, LangChain/LangGraph (Go port), OpenAI, Anthropic, Google Gemini, Ollama, gRPC, Redis, Cobra, Viper

Production-grade LLM pipeline primitives in Go — a port of LangChain, LangGraph, and LangSmith that replaces Python’s GIL-limited async model with native goroutines. Features entity extraction chains, multi-provider agent loops with tool use, HTTP workflow server, compile-time type safety via generics, sub-1ms startup.

Why it exists: Python LangChain is unsuitable for high-throughput inference pipelines (GIL-limited async, high memory, slow startup). bunshin-go solves the latency, memory, and concurrency limitations — same orchestration primitives, Go’s native scheduler.

pigeon-bomber#

Status: Active Development · Stack: Godot Engine, GDScript

A 2D physics-based action game developed with the Godot Engine. Features structured scene modularity (Main, Player, Background, Target, Explosion, Bomb), customized controls, and an integrated development pipeline equipped with Godot-Vim, Git, and advanced LLM plugins.

remiges-smtp#

Status: Open Source · Stack: Go, SMTP

High-performance SMTP client in Go with automated local file-scraping capability for dynamic transmission of payload data. Validates idiomatic Go concurrency patterns and robust error handling.

Planned Open Source#

akashic — Cloud Secrets Library#

Anime-inspired secrets management library. Reads secrets directly from AWS Secrets Manager and Azure Key Vault into pydantic-settings fields (Python) and struct tags (Go), eliminating manual environment variable re-storage.

  • Python: pip install akashic
  • Go: go get github.com/stlimtat/akashic
  • Field mapping: explicit annotation per field
